 # Process Monitoring Tool (Linux - C Language)

->A simple real-time process monitoring tool developed in C for Linux systems.
It displays CPU usage, memory usage, active processes, generates reports, and compares values with the Linux top command.

->Features
Display CPU Usage
Display Memory Usage
View Running Processes
Real-Time Monitoring
Generate System Reports
Compare with Linux top

->Installation & Compilation
gcc process_monitor.c -o monitor
./monitor
